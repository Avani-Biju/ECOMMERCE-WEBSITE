# E-Commerce Website

This project was created as part of my **Day 4 internship assignment**, where I continued learning Angular by building a simple e-commerce website.

The project focused on understanding Angular Routing, API integration, and reusable components while creating a multi-page application. Product information is fetched dynamically from the **Fake Store API**, allowing the application to display products instead of using hardcoded data.

The application includes multiple pages such as Home, Products, Product Details, and Contact, along with Bootstrap components and custom styling.

## ▶️ Running the Project

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
ng serve
```

Open your browser and visit:

```text
http://localhost:4200
```
